Welcome to My Portfolio!

I'm Adam Mhal, and this is my interactive portfolio. Think of it as a solar system of my work. Here's a quick guide to get you started:

  1. 1. Explore the system by dragging, panning, and zooming.
  2. 2. Click on any planet (or use the top navigation) to learn more about a section.
  3. 3. Press 'R' at any time to return to this main view.
  4. 4. If you're bored, try pressing the rocket button in the bottom right!

You can press 'T' at any time to see this tutorial again.
